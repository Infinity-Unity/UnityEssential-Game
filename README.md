## 🧩 About the Project

**Unity Essential Game** — это мой первый практический опыт разработки игры с использованием движка Unity.

В рамках проекта я реализовал базовую игровую механику, взаимодействие объектов и пользовательский интерфейс.
Основная цель — закрепить фундаментальные знания и понять структуру игрового проекта.

---

## 🎯 What I Learned

Во время разработки я изучил и применил на практике:

### 🔧 Unity Fundamentals

* Навигация и работа с **Unity Editor**
* Иерархия объектов (Hierarchy, Scene, Game view)
* Работа с **Prefab'ами** и компонентами

### 💻 Programming (C#)

* Написание и подключение скриптов
* Основы **объектно-ориентированного программирования**
* Работа с методами `Start()` и `Update()`
* Управление поведением объектов через код

### ⚙️ Game Mechanics

* Реализация управления игроком
* Обработка ввода (Input System)
* Создание игровой логики

### 🧱 Physics & Interaction

* Коллизии (**Collider**)
* Триггеры (**Trigger events**)
* Работа с **Rigidbody**

### 🖥️ UI Basics

* Добавление пользовательского интерфейса
* Отображение информации на экране
